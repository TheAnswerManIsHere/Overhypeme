### API server's /api/route-stats endpoint fails due to database issue.
The `GET /route-stats` handler in the API server attempts to query the `routeStatsTable`. If the database is in an inconsistent state (e.g., `route_stat_events` table is missing or corrupted due to migration issues), the database query fails, causing the API endpoint to return a 502.

```typescript
router.get("/route-stats", async (req, res) => {
  try {
    const n = Math.min(Number(req.query.n) || 3, 10);
    const rows = await db
      .select({
        routeKey: routeStatsTable.routeKey,
        visitCount: routeStatsTable.visitCount,
      })
      .from(routeStatsTable)
      .orderBy(desc(routeStatsTable.visitCount))
      .limit(n);
    res.json({
      routes: rows.map((r) => r.routeKey),
      stats: rows,
    });
  } catch (err) {
    req.log.warn({ err }, "route-stats: GET failed, returning empty list");
    res.json({ routes: [], stats: [] });
  }
});
```

*Analysis*: While the `GET` endpoint has a `try/catch` that returns an empty list on error, the `POST` endpoint (which is also failing with 502s according to breadcrumbs) does not explicitly return a 200/204 on error, which could lead to a 502 if the database operation throws an unhandled exception or if the connection pool itself is unhealthy. The `POST` handler also attempts to insert into `routeStatEventsTable`, which is the table introduced in the problematic migration.
(See @artifacts/api-server/src/routes/routeStats.ts)

### Browser receives 502 for module script, leading to TypeError.
If the web server itself is returning 502s for static assets (including JavaScript module chunks), the browser cannot parse the response as a valid module script. This results in the `TypeError: Importing a module script failed.`

*Analysis*: The 502 errors on `/api/route-stats` indicate a general instability or misconfiguration of the backend. If this instability extends to serving static assets, the browser will fail to load the main application bundle, leading to the observed `TypeError` at the root path.
