**Issue ID:** 7421183992
**Project:** sentry_project_backend
**Date:** 4/28/2026, 9:29:34 PM
## Issue Summary
Migration Failure: Column Already Exists in membership_history
**What's wrong:** Migration failed attempting to **add an existing column** to `membership_history`.
**In the trace:** The failure occurred during **database migration** execution within a transaction.
**Possible cause:** The migration script is **idempotency-impaired**; it runs an `ALTER TABLE ADD COLUMN` command that has already been executed.
